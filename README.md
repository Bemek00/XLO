# XLO

Prosty portal ogłoszeniowy inspirowany OLX. 

Umożliwia:
- przegląd ogłoszeń
- rejestrację konta
- logowanie
- dodawanie ogłoszeń
- podstawowe zarządzanie swoimi ogłoszeniami
